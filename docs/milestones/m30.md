# M30 — Futuristic Theme (Design System) — STYLING

> **Goal**: Build the complete visual identity for YoTop10 with extensive detail. This milestone covers ALL styling aspects - colors, typography, spacing, components, animations, and navigation for both themes.

> **IMPORTANT**: All styling has been deferred. The platform currently has NO custom styling. This milestone adds all visual design.

## Futuristic Theme - Component Specifications

### Buttons

**Primary Button**
- Border radius: 9999px (pill shape)
- Background: Brand Primary (#FF4500)
- Text: White, uppercase, tracked (letter-spacing: 0.1em)
- Hover: Slight scale (1.02), darker shade
- Active: Scale down (0.98)
- Shadow: None (flat design)
- Min height: 44px (touch target)

**Secondary Button**
- Border radius: 9999px (pill shape)
- Background: Transparent
- Border: 1px solid Border Accent
- Text: Brand Primary
- Hover: Background brand primary at 10% opacity

**Ghost Button**
- Border radius: 9999px
- Background: Transparent
- Text: Text Muted
- Hover: Text Primary, subtle background

### Cards (Post Cards, Profile Cards)

**Dark Mode Card**
- Background: Background Surface (#0A1120)
- Border: 1px solid Border Accent (rgba(255, 69, 0, 0.1))
- Border radius: 16px
- Padding: 24px
- Shadow: None (flat)
- Hover: Border opacity increases to 0.3

**Light Mode Card**
- Background: Background Surface (#FFFFFF)
- Border: 1px solid Border Accent (#E5E7EB)
- Border radius: 16px
- Padding: 24px
- Shadow: Subtle (0 4px 6px rgba(0,0,0,0.05))
- Hover: Border becomes Brand Primary

### Input Fields

- Background: Background Base
- Border: 1px solid Border Accent
- Border radius: 12px
- Padding: 14px 16px
- Focus: Border becomes Brand Primary, subtle ring
- Placeholder: Text Muted

### Navigation - Bottom (Mobile Only)

**Items**

## Theme Switching Behavior

### Retro Theme (DEFAULT)
- **DEFAULT theme** - loads automatically on first visit
- NO toggle available within the theme
- Fixed appearance (no dark mode)
- Users can switch to Futuristic theme if desired

### Futuristic Theme
- User can toggle between Light and Dark
- Toggle icon shows current state (sun for dark mode, moon for light mode)
- Clicking toggles to opposite
- Preference persisted in localStorage

### Switching Flow
1. On first visit, Retro theme loads automatically (DEFAULT)
2. User can switch to Futuristic via theme toggle
3. Theme switches instantly (no fade)
4. Preference saved
5. On next visit, saved preference applied

## Glassmorphism Specifications (Futuristic Theme - EVERYWHERE)

Glassmorphism is the CORE visual element of the Futuristic theme. It is applied to EVERY element to create that premium, floating, ethereal feel.

### Applied To:
- **All Navigation Bars** (Top bar, Bottom bar, Sidebar)
- **All Cards** (Post cards, Profile cards, Feature cards)
- **All Modals and Overlays**
- **All Input Fields**
- **All Buttons** (subtle glass effect)
- **Toast Notifications**
- **Dropdown Menus**
- **Slide-out Drawers**
- **Search Overlay**
- **User Profile Sections**
- **Comment Threads**
- **Sidebar Widgets** (Trending, Top Creators)

### Specifications
- **Background**: Background Surface at 60-80% opacity
- **Backdrop Blur**: 12px (desktop), 8px (mobile)
- **Border**: 1px subtle border using Border Accent color
- **Consistency**: Every element feels like it's floating on glass

### Visual Effect
- Elements appear to float above the background
- Creates depth and layering
- Subtle transparency shows underlying content
- Clean, modern, premium appearance

## Animation Specifications (Futuristic Theme - Non-Laggy)

Animations must be non-laggy - instantaneous response, no jank, smooth 60fps performance.

### Performance Requirements
- **No frame drops**: Maintain 60fps at all times
- **Instant response**: < 16ms between input and visual feedback
- **No layout shifts**: Animations must not cause content to jump
- **Hardware accelerated**: Use transform and opacity for animations

### Animation Types

| Animation | Duration | Easing | Trigger | Feel |
|-----------|----------|--------|---------|------|
| Button hover | 150ms | ease-out | Hover | Instant snap |
| Button click | 80ms | ease-in | Click | Tactile response |
| Card hover | 200ms | ease | Hover | Subtle lift |
| Modal open | 250ms | ease-out | Open | Smooth reveal |
| Modal close | 150ms | ease-in | Close | Quick dismiss |
| Toast slide | 200ms | ease-out | Show | Snappy notification |
| Page transitions | 150ms | ease | Navigation | Instant swap |
| Bottom nav tap | 100ms | ease | Tap | Immediate feedback |
| Input focus | 150ms | ease | Focus | Subtle glow |
| Dropdown open | 150ms | ease-out | Open | Quick expand |
| Drawer slide | 250ms | ease-out | Open | Smooth glide |
| Scroll reveal | 300ms | ease-out | Scroll | Gentle fade-in |

### Key Principles
- **No glow effects** (explicitly removed)
- **No particle animations**
- **No elaborate transitions**
- **Clean, minimal motion**
- **Purpose-driven**: Every animation has a function
- **Responsive**: Animations adapt to device capability
- **Accessible**: Reduced motion option available

### Interaction Feedback
- Every tap/click provides immediate visual feedback
- Hover states are crisp and responsive
- Loading states are clear and non-blocking
- Transitions feel natural and intuitive

To achieve an extreme futuristic feel, the following elements are implemented:

- Layered stacking order (z-index management)

### 2. Clean Lines
- Sharp, precise borders
- No rough edges
- Geometric shapes
- Consistent spacing

### 3. Premium Typography
- Modern geometric sans-serif fonts
- Clear hierarchy
- Readable at all sizes
- Professional appearance

### 4. Electric Accents
- Brand Primary (#FF4500) as the "electric" color
- Used sparingly for emphasis
- Creates visual interest without overwhelming

### 5. Depth & Layering
- Multiple layers of glassmorphism
- Overlapping elements where appropriate
- Clear visual hierarchy

### 6. Minimalist Approach
- Only essential elements shown
- Ample whitespace
- No clutter
- Focus on content

### 7. Smooth Interactions
- Every interaction has feedback
- Transitions are fluid
- No jarring movements

### 8. Consistent Design Language
- Uniform styling across all components
- Predictable behavior
- Coherent visual system

### 9. Mobile-Optimized
- Touch-first design
- Large tap targets
- Finger-friendly interactions

### 10. Retro Theme First (Default)
- Classic 2000s colors as default
